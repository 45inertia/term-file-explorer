# Commands List

In the first iteration of this app I am allowing the user to type commands in relevant to the
current working directory that they have in front of them. 

Initially I would like to have set commands myself, in future I would like to implement an
embedded terminal which would nullify any work done on this command entering process.

The `cd` command needs special handling as it needs to be intercepted by updating the 
`current_path_`. 

## List

- `cd`
- `ls`/`dir`
- `mkdir`
- `rm`/`del`
- `git status`?
